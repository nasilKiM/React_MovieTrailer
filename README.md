## 🍿 프로젝트 소개 

    <무비찾냥> 은 영화소개 웹사이트입니다.
    Moive Open API를 활용하여 최신영화 / 인기영화 / 개봉예정 영화의 목록을 조회할 수 있습니다. 또한 궁금한 영화를 검색하고 트레일러 영상을 볼 수 있습니다.

## 🚀 배포 주소

    serverless 배포
    https://react-movie-trailer-project.vercel.app/
